Extract PDF's

This tool is used to scrape the DOM for the URL's listed below to extract PDF information. This can be extended to extract any information from the DOM.
Use "jquery expression" only if you know what you are doing
If you have a specific use case for scraping PDF files, use jquery expression that needs to be run on the DOM loaded by the page. For example the default expression that is ran on the DOM is window.$('a[href*=".pdf"]').attr('href')

Make sure your expression returns the value from the selected DOM element
